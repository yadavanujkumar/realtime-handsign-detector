🚀 Quick Integration

1. Include the SDK

<script src="https://yourserver.com/static/js/handsign-sdk.js"></script>

2. Initialize the detector

const handDetector = new HandSignDetector({ serverUrl: 'https://yourserver.com', onDetection: (results) => { console.log('Hand signs detected:', results); // Handle detection results } });

3. Start detection with your video stream

// Get video element from your app const videoElement = document.getElementById('yourVideoElement'); // Start detection handDetector.startDetection(videoElement); // Stop detection handDetector.stopDetection();

4. Handle detection events

handDetector.on('thumbs_up', () => { console.log('User gave thumbs up!'); // Show reaction, send emoji, etc. }); handDetector.on('peace', () => { console.log('Peace sign detected!'); // Add peace emoji to chat });

📱 Platform Integration Examples

Zoom App Integration

// Zoom Apps SDK integration import zoomSdk from '@zoom/appssdk'; import { HandSignDetector } from './handsign-sdk.js'; const detector = new HandSignDetector({ serverUrl: 'your-server-url' }); // Get Zoom video stream zoomSdk.getVideoStream().then(stream => { detector.startDetection(stream); });

Microsoft Teams Integration

// Teams SDK integration import * as microsoftTeams from "@microsoft/teams-js"; import { HandSignDetector } from './handsign-sdk.js'; microsoftTeams.initialize(() => { const detector = new HandSignDetector({ serverUrl: 'your-server-url' }); // Integrate with Teams video detector.startDetection(videoElement); });

Google Meet Integration

// Chrome extension for Google Meet chrome.tabs.executeScript({ code: ` const detector = new HandSignDetector({ serverUrl: 'your-server-url' }); // Find Meet video element const videoElement = document.querySelector('video[autoplay]'); if (videoElement) { detector.startDetection(videoElement); } ` });

📖 API Documentation

REST API

POST /api/detect Content-Type: application/json { "image": "data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAAAQ..." } Response: { "success": true, "results": { "hands_detected": 1, "hand_signs": [ { "hand_index": 0, "sign": "thumbs_up", "confidence": 0.95 } ], "landmarks": [...] } }

WebSocket API

// Connect to WebSocket const socket = io('https://yourserver.com'); // Send frame for detection socket.emit('detect_frame', { image: 'data:image/jpeg;base64,...', timestamp: Date.now() }); // Receive results socket.on('detection_result', (data) => { if (data.success) { console.log('Detection results:', data.results); } });
